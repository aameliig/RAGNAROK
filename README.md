# RAGNAROK: Retrieval‑Augmented Generation Neutral Assessment & Reasoning Optimization Kit

> **From technical metrics to configurable business value**
<br>

**RAGNAROK is a сonfigurable modular evaluation framework for Retrieval-Augmented Generation (RAG) systems.**

We propose an evaluation framework for RAG pipelines that is **not a black box**. Instead, it is a
**transparent, modular, and user-configurable** system that bridges the gap between technical
performance and **business value**.

**RAGNAROK** can be run in **LLM-free** mode: Unlike existing evaluation libraries that rely on proprietary LLM judges, RAGNAROK can perform the entire evaluation pipeline **locally** using sentence embeddings, clustering algorithms, Natural Language Inference (NLI), and configurable business rules.

The framework combines **retrieval quality**, **generation quality**, and **business metrics** into a single configurable **Business Score (0–100)** while simultaneously providing automatic **root-cause diagnostics**.
